# Lineup Tracker

Een tool om speelminuten van jeugdvoetbalspelers bij te houden per wedstrijd, inclusief posities, tegenstanders en thuis/uit-info.

## Features

- Speelminuten per speler per wedstrijd
- Positie-tracking
- Thuis/uit aanduiding
- Tegenstander per wedstrijd
- Rating en score per wedstrijd
